1 How It Works

This tool decodes Base64 encoded images and displays them directly in your browser. No data is sent to any server - everything happens locally.

  • Paste Base64: Input your Base64 string with or without the data URI prefix
  • Auto-Detection: The tool automatically detects the image format from the data
  • Instant Preview: See your decoded image immediately in the preview area
  • Download Option: Save the decoded image to your device
Supported Formats

This viewer supports all common image formats including PNG, JPEG, GIF, WebP, SVG, and BMP. The format is automatically detected from the Base64 header or data URI prefix.
