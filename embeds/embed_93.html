1 Essential Shortcuts (Must Know!)

These are the basics that everyone should memorize. They work almost everywhere!

Ctrl + C Copy selected text or item
Ctrl + V Paste what you copied
Ctrl + X Cut selected text or item
Ctrl + Z Undo your last action
Ctrl + Shift + Z Redo (undo the undo)
Ctrl + A Select all text or items
Ctrl + F Find text on the page
Ctrl + P Print the current page
Ctrl + S Save the current page/document
Pro Tip: Paste Without Formatting

Use Ctrl + Shift + V to paste text without any weird formatting from websites!
