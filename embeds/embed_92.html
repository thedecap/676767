1 How It Works

This tool transforms your JavaScript code through a multi-step process to make it harder to read and understand while maintaining full functionality.

  • Comment Removal: All single-line (//) and multi-line (/* */) comments are stripped
  • Whitespace Trimming: Excess whitespace is removed for minification
  • Unicode Handling: Special characters are safely encoded for Base64 compatibility
  • Base64 Encoding: The cleaned code is encoded to Base64 format
  • Wrapper Generation: An eval wrapper is created that decodes and executes the code
Important Note

This is basic obfuscation intended to deter casual inspection. For production applications requiring serious protection, consider professional-grade obfuscation tools with additional features like variable renaming, control flow flattening, and dead code injection.
